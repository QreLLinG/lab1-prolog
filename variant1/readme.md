# Вариант 1 — Лабораторная работа 1

## Условие

Флэш — собака. Ровер — собака. Бутси — кошка. Стар — лошадь.  
Флэш чёрная. Бутси коричневая. Ровер рыжая. Стар белая.  

Домашнее животное — собака или кошка.  
Животное — домашнее животное или лошадь.  

У Тома есть собака не чёрного цвета.  
У Кейта есть лошадь или что-то чёрного цвета.

3. Определить владельцев чего-либо.
