# Лабораторная работа 1: Факты и правила в Prolog

## Описание

Репозиторий содержит решения трёх вариантов лабораторной работы по курсу «ISO-Prolog (SWI-Prolog)».

Каждый вариант включает:
- Файл с программой на Prolog (факты и правила) — код на английском.
- README на русском с условием, запросами, результатами и объяснением.
